📋 Registration Form Practice

A clean and modern Registration Form UI built using HTML5 and CSS3.
This project demonstrates the use of various HTML form elements, built-in validation, and basic CSS styling to create a simple and user-friendly registration form.

---

🚀 Features

- User registration form interface
- Input validation using HTML attributes
- Mobile number pattern validation (10 digits)
- Email input validation
- Password input field
- Age input with range restriction (18–100)
- Date of birth picker
- Search input field
- Gender selection using radio buttons
- Hobbies selection using checkboxes
- Bio textarea
- Profile picture upload option
- Submit and Reset buttons
- Clean and simple UI design

---

🛠 Technologies Used

- HTML5
- CSS3
- CSS Variables
- Flexbox Layout

⚙️ Form Validation

This form uses built-in HTML validation attributes such as:

- "required"
- "minlength"
- "maxlength"
- "pattern"
- "min"
- "max"

Example:

<input type="tel" pattern="[0-9]{10}" required>

This ensures the mobile number must contain exactly 10 digits.
